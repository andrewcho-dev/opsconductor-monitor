# Legacy Archive

This directory contains archived legacy code and documentation from OpsConductor v1.

**DO NOT USE** - This code is kept for reference only.

## Contents

- `legacy_backend/` - Original Flask backend with connectors
- `legacy_frontend/` - Original React frontend
- `legacy_docs/` - Documentation for legacy architecture
- `legacy_root/` - Root-level scripts and configs from v1

## Why Archived

OpsConductor v2 is a complete refactor with:
- Clean core architecture (7 files vs 200+)
- Database-driven addon system (no hardcoded connectors)
- FastAPI + Celery + Redis architecture
- Real-time updates via Socket.IO

## Active Development

All active development is now in:
- `backend/` (was backend_v2/)
- `frontend/` (was frontend_v2/)
- `docs/` (v2 documentation only)
